La fonction
<a href="https://p5js.org/reference/p5/scale">scale()</a>
modifie l’échelle du système de coordonnées actuel selon le facteur spécifié.

Les fonctions
<a href="https://p5js.org/reference/p5/push">push()</a>
et
<a href="https://p5js.org/reference/p5/pop">pop()</a>
permettent respectivement de sauvegarder et de restaurer le système de coordonnées.

Dans cet exemple, un carré de taille 200 est dessiné à l’origine, avec trois facteurs d’échelle différents.
