Cet exemple démontre l'utilisation de
<a href="https://p5js.org/reference/p5/loop">loop()</a>
et
<a href="https://p5js.org/reference/p5/noLoop">noLoop()</a>
pour mettre en pause et reprendre l'animation

Un clic de souris active ou désactive la boucle d'animation. Si la boucle d'animation
est arrêtée, l'utilisateur peut appuyer sur n'importe quelle touche pour avancer d'une image.
Note : l'utilisateur doit cliquer pour définir le focus sur la toile afin que 
les touches soient enregistrées.

Avancer d'une seule image se fait en appelant la fonction
<a href="https://p5js.org/reference/p5/redraw">redraw()</a>,
ce qui entraîne un seul appel à la fonction draw().
