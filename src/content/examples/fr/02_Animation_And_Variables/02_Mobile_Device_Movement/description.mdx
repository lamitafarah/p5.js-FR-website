La fonction <a href="https://p5js.org/reference/p5/deviceMoved" target="_blank">deviceMoved()</a>
 s'exécute lorsque l'appareil mobile affichant le croquis se déplace.
Dans cet exemple, la valeur de
<a href="https://p5js.org/reference/p5/accelerationX" target="_blank">accelerationX</a>,
<a href="https://p5js.org/reference/p5/accelerationY" target="_blank">accelerationY</a>,
et <a href="https://p5js.org/reference/p5/accelerationZ" target="_blank">accelerationZ</a>
définit la position et la taille d'un cercle.

Cela ne fonctionne que sur les appareils mobiles.
