En utilisant la méthode 
<a href="https://p5js.org/reference/p5/mask" target="_blank">mask()</a>,
vous pouvez créer un masque pour une image afin de spécifier la transparence
dans différentes parties de l'image.
Pour exécuter cet exemple localement, vous aurez besoin de deux fichiers image et d'un serveur 
local en fonctionnement. 
<a href="https://github.com/processing/p5.js/wiki/Local-server">local server</a>.
