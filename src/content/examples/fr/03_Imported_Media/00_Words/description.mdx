La fonction <a href="https://p5js.org/reference/p5/text" target="_blank">text()</a> est utilisé pour insérer du texte dans la toile.
Vous pouvez changer la police et la taille du texte en utilisant les fonctions <a href="https://p5js.org/reference/p5/loadFont" target="_blank">loadFont()</a>
et <a href="https://p5js.org/reference/p5/fontSize" target="_blank">fontSize()</a>.
Le texte peut être aligné à gauche, au centre ou à droite avec la fonction <a href="https://p5js.org/reference/p5/textAlign" target="_blank">textAlign()</a>,
et, comme pour les formes, le texte peut être coloré avec <a href="https://p5js.org/reference/p5/fill" target="_blank">fill()</a>.
