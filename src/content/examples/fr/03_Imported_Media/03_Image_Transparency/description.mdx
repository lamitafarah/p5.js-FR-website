Ce programme superpose une image à une autre en modifiant la valeur alpha de 
l’image à l’aide de la fonction 
<a href="https://p5js.org/reference/p5/tint" target="_blank">tint()</a>
Déplacez le curseur de gauche à droite sur la toile pour changer la position de l’image.
Pour exécuter cet exemple localement, vous aurez besoin d’un fichier image et d’un
<a href="https://github.com/processing/p5.js/wiki/Local-server"> serveur local </a> en fonctionnement.
