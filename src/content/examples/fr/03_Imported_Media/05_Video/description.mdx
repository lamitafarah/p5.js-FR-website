En utilisant les fonctions  <a href="https://p5js.org/reference/p5/noCanvas" target="_blank">noCanvas()</a>
et <a href="https://p5js.org/reference/p5/createVideo" target="_blank">createVideo()</a> 
vous pouvez intégrer une vidéo dans le
<a href="https://developer.mozilla.org/en-US/docs/Glossary/DOM" target="_blank">DOM</a> sans l’inclure sur la toile.
Pour intégrer une vidéo à l’intérieur de l’élément canvas, consultez l’exemple
<a href="https://p5js.org/examples/dom-video-canvas.html" target="_blank">Vidéo sur une toile</a>.
